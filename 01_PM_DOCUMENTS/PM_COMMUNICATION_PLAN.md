# 沟通管理计划

| doc_role | instance_status | init_behavior | fact_declaration |
|---|---|---|---|
| Project Document Template | template | 由 Project Manager 在规划阶段填充 | 无真实项目数据 |

- 版本：v0.1
- 状态：Draft
- 最后更新：YYYY-MM-DD

| 沟通项 | 频率/触发 | 参与人 | 输出 |
|---|---|---|---|
| Scope Baseline Review | 启动阶段一次 | Project Owner、PM、Product/Tech Owner | 审批记录、Decision、PU |
| Sprint Planning | 每 Sprint | Product Owner、PM、Tech | Sprint Plan |
| Daily Briefing | 工作日按需 | Project Owner、PM | Briefing/To-do |
| Risk Review | 每周或 High Risk 触发 | PM、Risk Owner、Project Owner | RAID 更新 |
| Sprint Review/Retro | 每 Sprint 末 | 全体相关角色 | Review/Retro |
| M1 Demo Review | M1 完成时 | Project Owner、PM、Tech/UAT | 验收记录 |
| P0 Release Review | 发布前 | Sponsor、PM、Tech、UAT | Release Decision |

所有会议必须生成纪要；所有实质更新先进入 Pending Updates。
