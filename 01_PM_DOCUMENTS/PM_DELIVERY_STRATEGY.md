- 版本：v0.1

## 模式

采用 Hybrid：外层以 PMBOK 启动/规划/执行/监控/收尾和 PRINCE2 阶段门禁控制；内层在批准的 Scope 内使用 1-2 周 Sprint。

## 阶段

1. 启动与规划：批准章程、需求、范围、角色、策略和估算方法。
2. M1 实现与验证：建立核心闭环。
3. P0 完整实现：补齐第 30.1 节剩余 P0。
4. P0 集成、跨平台与双 Agent 验收。
5. 发布准备与阶段收尾。

## 门禁

- Gate 1：Project Brief/Scope Approved。
- Gate 2：WBS、估算、测试计划 Approved。
- Gate 3：M1 Demo Human Accepted。
- Gate 4：P0 自动验收、Windows/macOS、Cursor/Codex 全部通过。
- Gate 5：发布清单与移交批准。

## 容差
