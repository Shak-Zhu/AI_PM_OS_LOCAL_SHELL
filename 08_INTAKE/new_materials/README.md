# 08/Intake — New Materials

本目录是可选的暂存区，用于接收用户导入的本地文件。

## 使用说明

将文件（.md, .docx, .xlsx, .pdf 等）放入本目录后，在对话中引用文件名即可。

## 登记

所有放入本目录的文件均应在 `00_PM_MEMORY/PM_INPUT_LOG.md` 中登记，`source_type` 为 `local_file`，`retrieval_method` 为 `file_read`。

## 注意事项

- 本目录不是系统唯一输入入口。
- 用户也可直接粘贴文本、上传附件、或提供 Cooper/浏览器 URL。
- 原始文件内容默认不复制到项目模板中，只登记元数据。
