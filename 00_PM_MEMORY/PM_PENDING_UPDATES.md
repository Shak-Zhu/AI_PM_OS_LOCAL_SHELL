## 待审批更新条目

> **状态说明**：以下为所有状态关键词。任何 PU 条目必须属于其中一种状态。

| 状态关键词 | 说明 |
|---|---|
| Proposed / 待审批 | 已提交，等待 Project Owner 审批 |
| Approved / 批准 | Project Owner 已批准，等待应用 |
| Rejected / 拒绝 | Project Owner 已拒绝，不应用 |
| Applied / 已应用 | 已写入正式文件并同步 JSON |

| PU ID | 更新对象 | 建议更新 | 来源 | 影响 | 状态 |
|---|---|---|---|---|---|

Project Owner 可逐项批准（Approved）、拒绝（Rejected）或要求修改。已批准更新须应用（Applied）后写入正式文件并同步 JSON。未批准（Proposed）的更新不得直接应用至正式文件。
